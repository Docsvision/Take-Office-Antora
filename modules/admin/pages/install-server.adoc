[#linux]
== Установка на Linux

При первоначальной установке на сервер Linux модуль устанавливается вместе с модулем {pl}.

[#windows]
== Установка серверной части на Windows

Серверная часть приложения _{to}_ должна быть установлена на машину Windows.

WARNING: Перед установкой ознакомьтесь с разделом "xref:ROOT:requirements.adoc[]".

Пользователь, выполняющий установку, должен обладать правами локального администратора.
// и являться _администратором {dv}_ быть членом группы "{dv} Administrators" на компьютере с сервером {dv}.

. Запустите пакет установки `{dv} TakeOffice server.msi`.
+
.Мастер установки серверной части приложения {to}
image::install-server-hello.png[Мастер установки серверной части приложения {to}]
+
. Примите условия лицензионного соглашения для продолжения установки.
+
.Условия лицензионного соглашения
image::install-server-license.png[Условия лицензионного соглашения]
+
. Нажмите на кнопку *Установить* и дождитесь завершения установки модуля.
+
.Мастер установки серверной части приложения {to}
image::install-server-confirm.png[Мастер установки серверной части приложения {to}]
+
. На следующем экране нажмите кнопку *Готово*, чтобы закрыть мастер установки.
+
